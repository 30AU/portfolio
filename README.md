# Minimal Mistakes GitHub Pages site

This repo is a minimal Jekyll site using the Minimal Mistakes theme via `remote_theme`.

How to use:
1. Create this repository (name it `30AU.github.io` for a user site) in GitHub web UI.
2. Add the files above using "Add file → Create new file".
3. Commit to the `main` branch.
4. In the repository Settings → Pages, set the source to `main` branch and `/ (root)` folder, then Save.
5. Wait a minute and visit https://30AU.github.io (or https://30AU.github.io/portfolio for a project site).

Notes:
- `jekyll-remote-theme` is supported by GitHub Pages, so GitHub will fetch and use Minimal Mistakes automatically.
- If you use a project site, set `baseurl: "/REPO-NAME"` in `_config.yml`.
- To customize the theme, edit `_config.yml` and add layout/content files via the web editor.
